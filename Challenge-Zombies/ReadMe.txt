Zombie Game Description

	- There are two types of agents.
		x Humans
		x Zombies
	- Zombies chase humans
	- Humans run from zombies
	- If a zombie touches a human, the human becomes a zombie
	- The player wins when all zombies are dead, and his score is calculated as the number of humans remaining.
	- R - red brick
	- G - glass
	- B - same as r brick
	- Z - zombie
	- @ - player
	- L - light brick
	- . - ignore
